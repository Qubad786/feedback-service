TrueNAS reviews API

API will allow to save, list and download UI reviews

Add a new review

  1. You'll use the POST method to the /api/reviews/add/ endpoint. Append your access token as a header in your request.
    Here's an example: <Authorization: Bearer {token}>

    Example call:
    POST http://localhost:8000/api/reviews/add/

    Body 
    {
        "url": "truenas.com/index"
        "review": "This is my firest",
    }
    Example Response: 
    {
        "review_id'": 1
        "success": True
    }

Get list of paginated reviews

You can use this API to get list of reviews.

  1. You'll use the GET method to the /api/reviews/ endpoint. Access token is not required.

    Example call:
    GET http://localhost:8000/api/reviews/?start=1&length=10

    Example Response: 
    {
        'draw': 1,
        'recordsTotal': '100',
        'recordsFiltered': '10',
        'data': [{
            "id": "1",
             "url": "https://truenas.com/1",
             "date_created": "2022-05-16 12:18:10.339572",
             "review": "Another review",
              ... 
        }] 
    }

Download csv of reviews

You can use this API to get download csv file of reviews.

  1. You'll use the GET method to the /api/reviews/export/ endpoint. Access token is not required.

    Example call:
    GET http://localhost:8000/api/reviews/export/

    Response will be a csv file.
